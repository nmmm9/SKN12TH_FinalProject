# 🔑 DdalKkak API 토큰 설정 가이드

## 1. Slack App 설정

### 단계 1: Slack App 생성
1. https://api.slack.com/apps 방문
2. "Create New App" → "From scratch"
3. App Name: "DdalKkak"
4. 워크스페이스 선택

### 단계 2: Bot Token 획득
1. "OAuth & Permissions" 메뉴
2. "Bot Token Scopes" 추가:0
   ```
   app_mentions:read
   channels:read
   chat:write
   files:read
   files:write
   users:read
   commands
   im:history
   im:read
   im:write
   ```
3. "Install to Workspace" 클릭
4. **Bot User OAuth Token 복사** (xoxb-로 시작)

### 단계 3: App-Level Token 생성
1. "Basic Information" → "App-Level Tokens"
2. "Generate Token and Scopes" 클릭
3. connections:write 스코프 추가
4. **App-Level Token 복사** (xapp-로 시작)

### 단계 4: Signing Secret 복사
1. "Basic Information" → "App Credentials"
2. **Signing Secret 복사**

### .env 파일 업데이트:
```env
SLACK_BOT_TOKEN=xoxb-YOUR-ACTUAL-BOT-TOKEN-HERE
SLACK_APP_TOKEN=xapp-YOUR-ACTUAL-APP-TOKEN-HERE
SLACK_SIGNING_SECRET=YOUR-ACTUAL-SIGNING-SECRET-HERE
```

## 2. Notion API 설정

### 단계 1: Notion Integration 생성
1. https://www.notion.so/my-integrations 방문
2. "Create new integration"
3. Name: "DdalKkak Integration"
4. **Internal Integration Token 복사** (secret_로 시작)

### 단계 2: 데이터베이스 설정
1. Notion에서 새 페이지/데이터베이스 생성
2. 데이터베이스 Share → Integration 연결
3. URL에서 Database ID 복사 (32자리 문자열)

### .env 파일 업데이트:
```env
NOTION_TOKEN=secret_YOUR-ACTUAL-NOTION-TOKEN-HERE
NOTION_DATABASE_ID=YOUR-DATABASE-ID-HERE
```

## 3. Jira API 설정

### 단계 1: Jira API Token 생성
1. https://id.atlassian.com/manage-profile/security/api-tokens
2. "Create API token"
3. **API Token 복사**

### .env 파일 업데이트:
```env
JIRA_URL=https://YOUR-DOMAIN.atlassian.net
JIRA_EMAIL=your-email@company.com
JIRA_API_TOKEN=YOUR-ACTUAL-JIRA-TOKEN-HERE
JIRA_PROJECT_KEY=YOUR-PROJECT-KEY
```

## 4. Google Meet API 설정

### 단계 1: Google Cloud Console 설정
1. https://console.cloud.google.com 방문
2. 새 프로젝트 생성: "DdalKkak"
3. Google Meet API 활성화
4. OAuth 2.0 클라이언트 ID 생성

### .env 파일 업데이트:
```env
GOOGLE_CLIENT_ID=YOUR-GOOGLE-CLIENT-ID-HERE
GOOGLE_CLIENT_SECRET=YOUR-GOOGLE-CLIENT-SECRET-HERE
```
